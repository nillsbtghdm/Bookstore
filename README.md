# Book Store Management System

A simple web-based book management system developed using Python, Flask, and SQLite.

## Features
- Add new books
- Search books by ISBN
- Delete books
- Display all books
- Store book information in an SQLite database
- Simple web interface using HTML templates

## Technologies Used
- Python
- Flask
- SQLite
- HTML

## Project Structure
```
book-store/
```

## Main Functionalities
- Create and manage a book database
- Perform CRUD operations (Create, Read, Delete)
- Search books by ISBN
- Display all books in the database

## Project Purpose
This project was developed as a university project to practice:
- Object-Oriented Programming (OOP)
- Working with databases using SQLite
- Building web applications with Flask
- Backend development fundamentals
